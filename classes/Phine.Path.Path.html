current()

current(boolean $env) : string

Returns the current working directory path.

The getcwd() function may return false if the parent of the current working directory is not readable. To workaround this issue, this method will check environment variables that may be set by the operating system for the current path.

Parameters

boolean $env

Only use the environment variables?

Throws

\Phine\Exception\Exception
